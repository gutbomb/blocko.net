What is Blocko?

What is Blocko?

Blocko is modern re-imagining of the Apple II Lego TC Logo interface card. Fully compatible with Lego Technic Interface A (Lego 9750) interface box to control all manner of machines built with Lego Technic bricks on vintage Apple II computers.

Why?

The original Lego TC interface card was never available on the retail market so over the years many of the cards provided to schools have been lost to time.

When?

After many false starts, lots of laziness, and finally a little bit of motivation, Blocko is available now!

Where?

Blocko is sold exclusively at Joe's Computer Museum. While you're there check out the rest of the inventory available there.

Compatibility

What computers does it work with?

Blocko works with the Apple IIe and Apple IIgs computer systems. The Blocko card needs to be installed in Slot 3.

Parts

Which Lego parts do I need?

To use Blocko you will need a Lego Technic Interface A (Lego 9750) with included cable (a standard IDC-20 cable) and 9 volt power supply, at least one Lego Technic 4.5v motor, and lots of Lego Technic bricks to build cool stuff. The LEGO Dacta: Technic Control Center (Lego 9700) is a great set that includes all of the motors, sensors, and bricks you will need to get started. The Lego Dacta: Basic Resource Set (Lego 9605) is a great set to add on to this for more options.

Software & Docs

How do I use this thing?

Blocko works with the Lego TC Logo software provided to schools in the 1980s and 1990s. This allows you to control the motors, lights, and sensors with the educational programming language LOGO.
